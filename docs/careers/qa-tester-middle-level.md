# QA Software Tester  

**Location:** Paradise Place, BTS Suan Luang Rama 9 (Hybrid 1–2 Days/Week)  
**Job Type** Full-time  

## About the Role  

We are looking for a detail-oriented **QA Software Tester** to join our engineering team. In this role, you will be responsible for ensuring the quality, reliability, and stability of our web and backend systems through thorough testing, documentation, and collaboration with developers and product teams.

You will work closely with software engineers, product owners, and designers to analyze requirements, design test cases, execute tests, and help deliver high-quality releases.

---

## Key Responsibilities
- Analyze business and technical requirements to create comprehensive test plans and test cases.
- Perform manual testing on desktop application, web applications, backend APIs, and system flows.
- Identify, document, and track bugs using issue-tracking tools (e.g., Jira).
- Verify fixes, retest issues, and perform regression testing.
- Collaborate closely with developers during development cycles (Agile/Scrum).
- Assist in defining acceptance criteria and improving QA processes.
- Perform basic API testing using tools like Postman or Swagger.
- Work with staging/production logs to identify issues.
- Support UAT preparation and test execution.
- Additional duties as directed by the supervisor or management.

---

## Requirements
- 2–5 years of experience in manual software testing.
- Strong understanding of testing techniques (functional, regression, integration, usability).
- Experience testing APIs (Postman, Swagger, or similar tools).
- Ability to analyze logs, perform root-cause exploration, and validate backend behavior.
- Experience working in Agile teams (Scrum/Kanban).
- Experience with test planning and leading testing cycles.
- Eagerness to learn and strong attention to detail.
- Good communication and teamwork skills.

---

## Nice-to-Have Skills
- Experience with automation testing (e.g., Playwright, Cypress, Selenium).
- Experience with performance or load testing tools.
- Experience testing mobile apps or embedded/IoT systems.
- Experience with SQL queries for testing data validation.
- Familiarity with CI/CD or dev tools (Git, pipelines).
- Basic understanding of cloud environments (AWS/Azure).
